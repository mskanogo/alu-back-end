# API Advanced

This project practices using the Reddit API with Python.

## Concepts

- API requests with requests module
- Pagination
- Recursion
- JSON parsing
- Sorting dictionaries

## Files

0-subs.py – Returns number of subscribers  
1-top_ten.py – Prints top 10 hot posts  
2-recurse.py – Recursively returns all hot post titles  
3-count.py – Recursively counts keyword occurrences
